Вам предстоит самостоятельно завершить спецификацию требований к ПО для второго релиза Chatty. Спецификацию нужно дополнить описанием функции карточек для запоминания новых слов. По пунктам это выглядит так: 
Написать Use Case для функции карточек (графическая и табличная форма);
Добавить функциональные требования к Use Case;
Дополните графический вариант Use Case, добавив на диаграмму варианты использования карточек для запоминания новых слов.
Опишите пользовательские требования для карточек в табличной форме. Шаблон таблицы уже есть в спецификации.
Опишите функциональные требования с применением уникальных текстовых меток, по аналогии с примером из спецификации.
Дополните графический вариант Use Case, добавив на диаграмму варианты использования функции просмотра видео.
Добавьте табличный вариант использования для функции просмотра видео. В спецификации должен появится минимум один ВИ с тремя расширениями базового сценария.
Опишите функциональные требования. Функциональные требования должны содержать уникальные текстовые метки.
Опишите атрибуты качества и бизнес-правила для новой функции просмотра видео, а также дополните глоссарий.
